//How the structure of the DB modal will look

Students Table:
---------------

* student_id (Primary Key)
* first_name
* last_name
* email
* phone_number
* date_of_birth
* address
* guardian_name
* guardian_contact

Instructors Table:
------------------

* instructor_id (Primary Key)
* first_name
* last_name
* email
* phone_number

ClassSchedule Table:
--------------------

* schedule_id (Primary Key)
* course_id (Foreign Key references Courses)
* day_of_week
* start_time
* end_time
* room_location

Courses Table:
--------------

* course_id (Primary Key)
* course_name
* description
* start_date
* end_date
* instructor_id (Foreign Key references Instructors)

Enrollments Table:
------------------

* enrollment_id (Primary Key)
* student_id (Foreign Key references Students)
* course_id (Foreign Key references Courses)
* enrollment_date

Payments Table:
---------------

* payment_id (Primary Key)
* student_id (Foreign Key references Students)
* course_id (Foreign Key references Courses)
* payment_date
* amount
* payment_method
